# LS100: Computational Behavioral Sciences 
## Foundations of Data Science & Applied AI/ML for Behavioral Sciences

Welcome to the comprehensive repository for **LS100**. This collection contains learning materials from Python fundamentals to advanced Data Science, Computer Vision, and Machine Learning models for deriving meaningful insights from complex behavioral data obtained from animals and humans. The content of this repo is used to teach LS100, a 4-credit research-based course at Harvard University. All content of this repo is open for any non-commercial usage; feel free to use the materials for learning and non-commercial teaching.

## Tech Stack

| Category | Tools & Libraries |
